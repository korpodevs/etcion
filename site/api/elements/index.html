    Elements¶

    pyarchi.metamodel.elements ¶

    Abstract element subclasses for the ArchiMate 3.2 Generic Metamodel.

    Defines the structure-element and behavior-element branches of the EPIC-004 hierarchy. All classes are abstract (_type_name not implemented) and cannot be instantiated directly.

    Reference: ADR-016, FEAT-04.1, FEAT-04.2; ArchiMate 3.2 Specification, Section 4.
